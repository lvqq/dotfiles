## My dotfiles config
Here's my dotfiles config base mostly on zsh

## Usage
```bash
# 1. clone the repo
git clone https://github.com/lvqq/dotfiles.git
# 2. enter the project
cd dotfiles
# 3. delete the directory of module u don't want to install
# 4. run bootstrap
sh bootstrap.sh
```
